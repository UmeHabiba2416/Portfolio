# Ume Habiba - Portfolio Website

![Portfolio Preview](https://umehabiba2416.github.io/Portfolio/)

**A modern, responsive portfolio website showcasing my work as a Full Stack Developer & UI/UX Designer.**

🔗 **[Live Demo](#)** (https://umehabiba2416.github.io/Portfolio/)

---

## 🌟 Features
- Modern UI/UX Design: Clean, intuitive, and visually appealing interface.
- Fully Responsive: Works on all devices (mobile, tablet, desktop).
- Dark/Light Mode: Toggle between themes for better user experience.
- Interactive Elements: Smooth animations, hover effects, and 3D card tilts.
- Project Showcase: Highlight your best work with detailed project cards.
- Experience Timeline: Showcase your professional journey and achievements.
- Contact Form: Easy way for visitors to get in touch (powered by EmailJS).
- Custom Cursor: Unique cursor effects for a premium feel.

## 🛠 Technologies Used
HTML5, CSS3, JavaScript, CSS Grid, Flexbox, EmailJS.
